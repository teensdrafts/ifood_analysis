# 📊 Маркетинговый анализ iFood

## 📝 Описание

В рамках данного проекта я выступаю в роли продуктового/маркетингового аналитика. По результатам последних маркетинговых кампаний было выявлено, что их эффективность оказалась ниже ожидаемой.

**Цель проекта** — провести комплексный анализ клиентских данных, выявить ключевые факторы, влияющие на покупательское поведение, и на основе полученных инсайтов предложить рекомендации по повышению эффективности будущих маркетинговых кампаний.

---

## 🔍 Выполненные этапы

### 📌 Подготовка данных

* Анализ типов данных и корректности форматов
* Обработка пропущенных значений
* Проверка и приведение названий столбцов к единому стилю

---

### 📊 Часть №1: Разведочный анализ (EDA)

* Обработка выбросов и пустых значений
* Создание новых признаков на основе существующих
* Корреляционный анализ для выявления зависимостей и аномалий
* Первичная постановка гипотез для последующего анализа

---

### 📈 Часть №2: Статистический анализ

В рамках статистического анализа были проверены следующие гипотезы и исследованы вопросы:

* Какие факторы оказывают наибольшее влияние на покупки в заведениях?
* Лидируют ли США по общему числу покупок?
* Верно ли, что клиенты, потратившие за последние два года на продукцию категории *Gold* сумму выше средней, совершают больше покупок в заведениях?
* Существует ли взаимосвязь между клиентами категории *Married & PhD* и объемом трат на рыбную продукцию?
* Какие факторы в наибольшей степени влияют на траты на рыбную продукцию?
* Зависит ли успех маркетинговой кампании от географического положения клиента?

Для проверки гипотез использовались соответствующие статистические критерии и методы оценки значимости, а также регрессионный анализ.

---

### 🤖 Часть №3: Кластерный анализ (сегментация клиентов)

Дополнительно был проведён кластерный анализ с целью сегментации клиентской базы.

Клиенты были разделены на группы на основе трёх ключевых признаков:

* уровень дохода
* суммарные траты в компании
* количество детей

Сегментация позволила:

* выявить поведенческие паттерны внутри групп
* определить наиболее ценные клиентские сегменты
* сформировать основу для персонализации маркетинговых кампаний
* предложить стратегии взаимодействия с различными сегментами

---

### 📉 Часть №4: Визуализация данных

В рамках визуализации были представлены ответы на следующие вопросы:

* Какая маркетинговая кампания оказалась наиболее успешной?
* Как выглядит портрет среднестатистического клиента iFood?
* Какие продукты пользовались наибольшим спросом?
* Какие виды покупок оказались наименее популярными?
